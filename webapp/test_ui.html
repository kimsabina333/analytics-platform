Analytics Platform
Инструмент для мониторинга платёжных метрик и AI-анализа. Помогает находить проблемы, понимать причины и принимать обоснованные решения — без необходимости разбираться в статистике.
Модули платформы
📊 Model Interpreter
Байесовский мониторинг Success Rate — процента успешных первых платежей. Модель знает «нормальный» SR для каждого сегмента и сигнализирует, когда реальный SR падает ниже ожидаемого диапазона.
Dashboard — обзор всех сегментов, алерты в реальном времени
Segments — детальный анализ по 12 параметрам (MID, geo, offer и др.)
AI Chat — вопросы на русском/английском, графики по запросу
💳 Payment Adviser
Мониторинг риск-метрик по каждому MID. Отслеживает Chargeback Rate, Fraud Rate и VAMP Rate с порогами Visa — превышение грозит штрафами или блокировкой MID.
Risk Metrics — динамика CB / Fraud / VAMP по месяцам
AI Chat — автоматический анализ аномалий и план действий
📈 Marketing Adviser
Анализ эффективности маркетинговых каналов. ROI, CAC, CPM и GP по источникам (Facebook, Google, TikTok, ADQ) — с AI-анализом для перераспределения бюджета.
ROI Dashboard — сводка по источникам: spend, LTV, ROI, CAC
AI Chat — оптимизация каналов и рекомендации по бюджету
Как работать с платформой
1
Откройте Dashboard — красные карточки означают активные алерты: SR упал ниже ожидаемого модельного диапазона. Кликните на карточку, чтобы увидеть график с доверительным интервалом.
2
Разберите причины через AI Chat — спросите "Почему упал SR у Google?" или "Сравни MID по SR". ИИ сам вызовет нужные инструменты, покажет данные и даст рекомендации.
3
Следите за риск-метриками — Payment Adviser → Risk Metrics. CB Rate и VAMP Rate выше 0.9% — это порог Visa Standard Program. Превышение → штрафы, в худшем случае закрытие MID.
4
Получите план действий — Payment Adviser → AI Chat автоматически анализирует текущие метрики при первом открытии и предлагает конкретные шаги по снижению рисков.
Глоссарий
SR Success Rate — % успешных попыток оплаты
CI Доверительный интервал — ожидаемый диапазон SR по модели
Alert SR упал ниже нижней границы CI — значимое отклонение
MID Merchant ID — платёжный процессор (Adyen, Checkout и др.)
CB Rate Chargeback Rate = споры / транзакции. Порог Visa: >0.9%
VAMP Visa Acquirer Monitoring Program — CB+Fraud только Visa-карты
LTV Lifetime Value — прогноз выручки с клиента за весь срок
ARPPU Средний доход с платящего пользователя (first + upsell)
